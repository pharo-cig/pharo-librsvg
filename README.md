This are [Pharo](https://github.com/pharo-project/pharo) bindings for the [librsvg](https://github.com/linebender/resvg) 
library, generated using [pharo-cig](https://github.com/estebanlm/pharo-cig).  

The generation recipe can be found at the class comment of [LibRsvg class](https://github.com/pharo-cig/pharo-librsvg/blob/main/src/RSVG/LibRsvg.class.st).
